# 🚀 Quick Start Guide

Get your Workout Tracker app running in 5 minutes!

## ⚡ Fast Setup

### 1. Open in Android Studio
```bash
File > Open > Navigate to WorkoutTrackerAndroid folder > OK
```

### 2. Wait for Gradle Sync
- Android Studio will automatically sync dependencies
- This may take 2-3 minutes on first run
- Watch the bottom status bar for progress

### 3. Run the App
- **Option A:** Click the green ▶️ button in the toolbar
- **Option B:** Press `Shift + F10`
- **Option C:** Menu: Run > Run 'app'

### 4. Choose Device
- **Emulator:** Create one via Device Manager if needed (Pixel 5, API 34 recommended)
- **Physical Device:** Enable USB debugging in Developer Options

### 5. Done! 🎉
The app will install and launch automatically

## 📱 First Launch

When you first open the app, you'll see:
- 3 sample workouts (Push Day, Pull Day, Leg Day)
- A "Start Workout" button to begin
- Options to create new workouts

## 🎯 Common Actions

### Create Your First Workout
1. Tap the **+** floating button (bottom right)
2. Enter "My Workout" as the name
3. Tap **+ Add Exercise**
4. Fill in exercise details
5. Tap **SAVE**

### Start a Workout Session
1. Tap **"Start Workout"** on home screen
2. Select a workout
3. Enter reps and weight for each set
4. Watch the rest timer count down
5. Complete all exercises
6. Tap **FINISH**

### Customize Settings
1. Tap ⚙️ Settings icon (top right on home)
2. Change weight unit (LBS/KG)
3. Enable/disable rest timer
4. Choose light/dark theme

## 🐛 Troubleshooting

### Gradle Sync Failed
```bash
File > Invalidate Caches > Invalidate and Restart
```

### App Won't Build
1. Check you're using JDK 17+: File > Project Structure > SDK Location
2. Update Android SDK to API 34: Tools > SDK Manager
3. Clean project: Build > Clean Project
4. Rebuild: Build > Rebuild Project

### Emulator Issues
- Create new emulator: Tools > Device Manager > Create Device
- Use Pixel 5 or newer with API 34
- Allocate at least 2GB RAM to emulator

## 🔥 Pro Tips

1. **Swipe Between Exercises** - Use ← → arrows during workout
2. **Skip Rest Timer** - Tap "Skip Rest" if you're ready to continue
3. **Reorder Exercises** - Use ↑↓ buttons when editing workouts
4. **Quick Edit** - Long press ⋮ on workout card for menu
5. **Delete Sets** - Tap "Skip Set" if you couldn't complete it
